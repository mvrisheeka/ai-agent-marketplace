Overview

The Resume Analyzer agent takes a candidate's resume and optionally a job description, then returns a structured evaluation covering skill match, formatting quality, missing keywords, and an overall fit score.

It's designed for both job seekers who want quick feedback and recruiters who want a first-pass filter before manual review.

PDF / DOCX / plain text JD-aware scoring ATS keyword check Section-level feedback

How it works

1
Upload Resume

Paste resume text or upload a file. The agent extracts sections automatically.

2
Provide Job Description (optional)

Paste the JD to enable role-specific keyword and skill-gap analysis.

3
Analysis runs

The agent scores each section — skills, experience, education, formatting — and checks ATS compatibility.

4
Get structured feedback

Receive a scored report with section-level suggestions and a final fit score (0–100).

Capabilities

  • Scores resume quality on a normalized 0-100 scale.
  • Extracts key skills from the provided profile text.
  • Generates practical summary and feedback for improvement.
  • Returns structured breakdown fields for UI rendering.

Inputs

Required

  • resume_text — Full resume content as plain text, or an uploaded PDF/DOCX.

Optional

  • job_description — Target JD text. Enables role-specific scoring and keyword gap detection.
  • mode — general (default) or jd_match. Auto-selected if a JD is provided.

Outputs

FIT SCORE

0–100 numerical score. Above 75 = strong match, 50–74 = moderate, below 50 = needs work.

SECTION SCORES

Individual scores for Skills, Experience, Education, Formatting, and Keywords.

MISSING KEYWORDS

List of terms present in the JD but absent from the resume, ranked by importance.

SUGGESTIONS

Actionable bullet-point feedback per section — what to add, reword, or remove.

Tips for best results

Tip: Always provide the job description. Without it, scoring is generic and misses role-specific nuance.
  • Use plain text resumes for most accurate parsing. PDF columns can confuse extraction.
  • Keep your resume under 2 pages — the agent flags length issues as a formatting penalty.
  • Run the analyzer after every major resume edit to track your score over time.
